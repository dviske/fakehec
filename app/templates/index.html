Welcome to FakeHEC!

FakeHEC is a simple service emulating the Splunk HTTP Event Collector allowing you to quickly setup and debug a collector for your application.

FakeHEC was developed by dviske and the source code is available at https://github.com/dviske/fakehec.

Getting started

To get started, please create a collector. A collector has a password associated with it so it cannot be seen by others (though should by no means store sensitive information in FakeHEC) and you will be given a token to pass to your application.

You can use this form both to create and to sign into your collector.
